Catalogo API


L'applicazione dispone di servizi API in stile REST che permettono ad altre applicazioni o script di accedere ai valori di particolato registrati dalle centraline Luftdaten. Il servizio permette l'accesso ai valori di particolato, sia realtime sia storici, per una località desiderata, grazie ad un paradigma di URL statico illustrato di seguito. Eventuali errori nell'utilizzo delle API vengono segnalati attraverso codici di risposta HTTP. Per tutte le richieste e le risposte dall'API viene utilizzato il formato JSON.

Di seguito è mostrato il paradigma delle API e delle relative informazioni associate.
